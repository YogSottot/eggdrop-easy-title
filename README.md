  Автор: tvrsh

Описание: Скрипт "простого" показа тайтлов вебстраничек.



 Установка: 
         1. Скопируйте скрипт etitle.tcl в папку scripts вашего бота.
         2. В файле eggdrop.conf впишите строку source scripts/etitle.tcl 
         3. Сделайте .rehash боту.
